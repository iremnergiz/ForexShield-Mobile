# Forex Shield: Mutfak Maliyet & Risk Analiz Paneli 📊

Bu uygulama, döviz bazlı gıda alımlarında (örneğin otel mutfakları için et alımı) kur dalgalanmalarının bütçe üzerindeki etkisini anlık olarak takip eden bir **Karar Destek Sistemi** örneğidir.

# Öne Çıkan Özellikler
* **Canlı Veri Akışı:** Uygulama her açıldığında döviz kurlarını internet üzerinden anlık olarak çeker.
* **Bütçe Risk Analizi:** 5.000 USD tutarındaki borcun, değişen kurlar karşısında belirlenen mutfak bütçesini aşıp aşmadığını otomatik analiz eder.
* **Görsel Uyarı Sistemi:** Bütçe aşıldığında kullanıcıyı renkli bir dashboard ile uyarır.

# Teknik Altyapı
* **Frontend:** Flutter (Dart)
* **API Bağlantısı:** HTTP üzerinden JSON veri işleme (Döviz verileri)
* **Mimari:** İş analizi odaklı modern dashboard tasarımı.
